DATA & FILE OVERVIEW

File List: <list all files (or folders, as appropriate for dataset organization) contained in the dataset, with a brief description>

1. Demographics.csv: contains the demographic data of the participants
2. BrakeResponseTime-Data.csv: contains the driving behavior data from the 3 brake response time trials namely BRT1, BRT2 and BRT3
3. BRT-criticalDHW-Data.csv: contains the safe/unsafe distance headway maintenance driving data (% of the time) from the 3 brake response time trials namely BRT1, BRT2 and BRT3
4. OcclusionTrials-criticalDHW-Data.csv: contains the safe/unsafe distance headway maintenance driving data (% of the time) from the two occlusion trials
5. Multilevel-Analysis-BRT-Data.csv: contains the data of linear mixed model for brake response time and headway distance at the moment lead car starts braking in the 3 BRT trials. 
6. OcclusionTrials-AttentionPercentage-Data.csv: contains the attentive and inattentive (% of the time) driving performance data during the two occlusion trials.

Demographics.csv

- Number of variables: 6
- Number of cases/rows: 32
- Variable List: ID (participant id), Sex (gender, values: M - Male, F - Female), Age (participant age - in years), Driving experience (in years), Self-estimated driving per year (in kilometers), Self-estimated life-time driving (in kilometers)

BrakeResponseTime-Data.csv

- Number of variables: 8
- Number of cases/rows: 96
- Variable List: ID (participant id), Trial (BRT1, BRT2, BRT3), Break reaction time (brake response time - in seconds), Perceived Headway Distance - Max (in meters), Perceived Headway Distance - Min (in meters), Perceived Headway Distance - Mean (in meters), Perceived Headway Distance - Std Dev (in meters), Perceived Headway Distance - 15th Percentile (in meters)

BRT-criticalDHW-Data.csv

- Number of variables: 7
- Number of cases/rows: 32
- Variable List: ID (participant id), BRT1 - Acting safe (%), BRT1 - Acting unsafe (%), BRT2 - Acting safe (%), BRT2 - Acting unsafe (%), BRT3 - Acting safe (%), BRT3 - Acting unsafe (%)

OcclusionTrials-criticalDHW-Data.csv

Multilevel-Analysis-BRT-Data.csv
